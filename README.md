# 🍕 Pizza Sales Analysis – Power BI Dashboard

A business intelligence dashboard built in Power BI to analyze pizza sales data. This project focuses on uncovering key insights about customer preferences, product performance, and sales trends to support data-driven decisions in a food business.

## 📊 Key Insights

- 💰 **Total Revenue**, **Average Order Value**, and **Best-Selling Pizza Types**  
- 📅 Sales performance across **days, weeks, and months**  
- 🕒 **Peak Order Times** and customer behavior patterns  
- 🍕 Analysis of **pizza categories, sizes**, and **most profitable items**  
- 📌 Identification of underperforming products and sales opportunities

---

## 🛠 Tools & Technologies

- **Power BI** for dashboard development and visualization  
- **Power Query** for data cleaning and transformation  
- **DAX** for calculated fields and KPIs  
- **Excel / CSV** for the data source
